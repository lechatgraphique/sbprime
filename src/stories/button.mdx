# Button

### Import
```
import { ButtonModule } from 'primeng/button';
```

### Directive
Un Button peut également être utilisé comme directive en utilisant `pButton`.
Contrairement au composant `p-button`, la directive `pButton` n'utilise pas d'effet, utilisez la directive `pRipple` pour activer l'ondulation.

### Accessibility
Le composant `Button` restitue un élément de bouton natif qui inclut implicitement tout accessoire transmis.
Le texte décrivant le bouton est défini avec `aria-label`.
S'il n'est pas présent,`label` est utilisé comme valeur.
Si le bouton est uniquement une icône ou si un modèle personnalisé est utilisé,
il est recommandé d'utiliser `aria-label` pour que les lecteurs d'écran puissent lire correctement l'élément.

```
<p-button icon="pi pi-check" aria-label="Submit" />
<p-button icon="pi pi-check" label="Submit" />

<p-button class="youtube p-0" aria-label="Youtube">
  <i class="pi pi-youtube px-2"></i>
  <span class="px-3">Youtube</span>
</p-button>
```
